### 代码

``` java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> st = new Stack<>();
        char[] cs = s.toCharArray();
        for (char c : cs) {
            if (st.isEmpty())
                st.push(c);
            else if (c == ')' && st.peek() == '(')
                st.pop();
            else if (c == ']' && st.peek() == '[')
                st.pop();
            else if (c == '}' && st.peek() == '{')
                st.pop();
            else
                st.push(c);
        }
        return st.isEmpty();
    }
}
```

非常经典的栈的应用，括号匹配问题。
